# Registro de Conflicto Git

## Archivo afectado

index.html

---

## Descripción del conflicto

Se generó un conflicto al intentar realizar un merge entre las ramas:

- conflicto-navbar
- conflicto-header

Ambas ramas modificaban la misma línea del archivo index.html:

```html
<h1>Eventos Chiapas</h1>


Cambios realizados:

Rama conflicto-navbar
<h1>Eventos Chiapas Oficial</h1>
Rama conflicto-header
<h1>Eventos Chiapas 2026</h1>

Solución aplicada:

El conflicto fue resuelto manualmente dejando la siguiente línea:

<h1>Eventos Chiapas Oficial 2026</h1>
