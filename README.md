Descripción del Proyecto:

Eventos Chiapas es un sitio web estático desarrollado como práctica colaborativa para aplicar el uso de Git y GitHub en un entorno de trabajo real. El proyecto consiste en crear una página web básica para una agencia local de eventos, incluyendo una página principal, una sección de eventos y una página de contacto.
Durante el desarrollo se implementan buenas prácticas de control de versiones, trabajo colaborativo mediante ramas, commits descriptivos, pull requests, resolución de conflictos y documentación del proyecto. Además, el sitio será publicado utilizando GitHub Pages para su acceso en línea.

Objetivo General:

Aplicar el uso de Git y GitHub en un entorno colaborativo realista mediante el desarrollo de un sitio web estático, utilizando buenas prácticas de control de versiones, documentación y trabajo en equipo.
Objetivos Específicos
Crear y administrar un repositorio en GitHub.
Implementar una estructura básica de un sitio web con HTML, CSS y JavaScript.
Utilizar ramas para dividir el trabajo entre los integrantes del equipo.
Realizar commits descriptivos para llevar un control adecuado de cambios.
Aplicar pull requests y revisiones de código colaborativas.
Simular y resolver conflictos de versiones utilizando Git.
Documentar correctamente el proyecto mediante archivos README.md y CONTRIBUTING.md.
Publicar el sitio web utilizando GitHub Pages.
